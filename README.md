# Codemao-Scrapy

编程猫爬虫

## Usage

```sh
# Get a workshop's information.
$ Codemao-Scrapy workshop <Workshop ID>
# Get an user's information.
$ Codemao-Scrapy user <User ID>
# Start an API Server
$ Codemao-Scrapy server <port>
```
